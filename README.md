# HFR RedFlag

Userscript Greasemonkey/Tampermonkey pour [forum.hardware.fr](https://forum.hardware.fr) qui met en evidence les posts ayant ete alertes a la moderation.

## Fonctionnalites prevues

- Detection des posts alertes
- Indicateur visuel : fond rouge et/ou icone sur les posts concernes

## Installation

1. Installer [Tampermonkey](https://www.tampermonkey.net/) ou [Greasemonkey](https://www.greasespot.net/)
2. Cliquer sur `hfr-redflag.user.js` puis "Raw" pour installer le script
